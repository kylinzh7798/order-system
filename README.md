# 🛒 Order System (Spring Boot)

A simple full-stack web application for managing products (CRUD), built with Spring Boot, Thymeleaf, and MySQL.

---

## 🚀 Features

- View product list
- Add new product
- Edit product (name & price)
- Delete product
- Login / Logout (Spring Security)
- Simple UI with Bootstrap

---

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA (Hibernate)
- Thymeleaf
- MySQL
- Spring Security
- Bootstrap 5
